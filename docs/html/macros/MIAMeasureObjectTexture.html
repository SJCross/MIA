MIA_MeasureObjectTexture

Description

Measure object texture across the image. Calculates the Haralick features. If "calibratedOffset" is true offsets are in calibrated units, otherwise units are in pixels.

Parameters

String objectsName
  • String imageName
  • int xOffset
  • int yOffset
  • int zOffset
  • boolean calibratedOffset
  • boolean showResults
