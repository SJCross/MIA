MIA_MeasureObjectOverlap

Description

Calculates the total voxel and percentage overlap between objects. Can be restricted to only comparing objects present in the same timepoint.

Parameters

String objectsName1
  • String objectsName2
  • boolean linkInSameFrame
  • boolean showResults
