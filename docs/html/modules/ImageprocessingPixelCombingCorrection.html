Combing correction

Description

Applies an integer pixel row shift to every other row (starting with top-most row).

Parameters

  • Input image
    • Description: Image to which the correction will be applied.
    • Type: InputImageP

  • Apply to input image
    • Description: Set to "true" to apply correction to the input image or "false" to store the corrected image separately.
    • Type: BooleanP
    • Default value: true

  • Output image
    • Description: Output image with correction applied.
    • Type: OutputImageP

  • Offset (px)
    • Description: Pixel offset to be applied to every other row.
    • Type: IntegerP
    • Default value: 0
