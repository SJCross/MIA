Convert 3D stack (switch Z and T)

Description

Checks if there is only 1 frame, but multiple Z-sections. In this case, the Z and T ordering will be switched

Parameters

  • Input image (default = "")

  • Apply to input image (default = "true")

  • Output image (default = "")

  • Mode (default = "Output timeseries")
