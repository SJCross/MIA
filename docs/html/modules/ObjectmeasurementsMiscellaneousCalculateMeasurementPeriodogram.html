Calculate measurement periodogram

Description

null

Parameters

  • Track objects

  • Spot objects
    • Type: ChildObjectsP
  • Measurement
    • Type: ObjectMeasurementP
  • Reporting mode
    • Type: ChoiceP
    • Default value: Key frequencies
    • Choices:
      • Key frequencies
      • Whole spectrum
  • Number of peaks to report
    • Type: IntegerP
    • Default value: 3
  • Number of bins
    • Type: IntegerP
    • Default value: 256

    • Type: ChoiceP
    • Default value: Skip measurement
    • Choices:
      • Skip measurement
      • Set points to mean
      • Set points to zero
