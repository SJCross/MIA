Calculate measurement periodogram

Description

Parameters

  • Track objects
    • Type: InputObjectsP

  • Spot objects
    • Type: ChildObjectsP

  • Measurement
    • Type: ObjectMeasurementP

  • Reporting mode
    • Type: ChoiceP
    • Default value: Key frequencies
    • Choices:
      • Key frequencies
      • Whole spectrum

  • Number of peaks to report
    • Type: IntegerP
    • Default value: 3

  • Number of bins
    • Type: IntegerP
    • Default value: 256

  • Missing point handling
    • Type: ChoiceP
    • Default value: Skip measurement
    • Choices:
      • Skip measurement
      • Set points to mean
      • Set points to zero
