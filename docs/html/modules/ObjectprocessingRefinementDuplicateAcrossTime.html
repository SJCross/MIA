Duplicate objects across time

Description

Creates a copy of objects across all frames in the specified image stack.

Parameters

  • Input objects
    • Type: InputObjectsP

  • Output objects
    • Type: OutputObjectsP

  • Coordinates storage mode
    • Type: ChoiceP
    • Default value: Common
    • Choices:
      • Common
      • Duplicate

  • Start frame mode
    • Type: ChoiceP
    • Default value: Fixed value
    • Choices:
      • Fixed value
      • Image measurement

  • Start frame fixed value
    • Type: IntegerP
    • Default value: 0

  • Start frame image
    • Type: InputImageP

  • Start frame image measurement
    • Type: ImageMeasurementP

  • Start frame offset
    • Type: IntegerP
    • Default value: 0

  • End frame mode
    • Type: ChoiceP
    • Default value: Fixed value
    • Choices:
      • Fixed value
      • Image measurement

  • End frame fixed value
    • Type: IntegerP
    • Default value: 0

  • End frame image
    • Type: InputImageP

  • End frame image measurement
    • Type: ImageMeasurementP

  • End frame offset
    • Type: IntegerP
    • Default value: 0
