Measure specific width

Description

Measures the width of the object along the axis passing through the centroid and specified point.

Parameters

  • Input objects
    • Type: InputObjectsP

  • Reference mode 1
    • Type: ChoiceP
    • Default value: Object centroid
    • Choices:
      • Object centroid
      • Image measurement
      • Object measurement

  • Reference image 1
    • Type: InputImageP

  • X-pos. image meas. 1 (px)
    • Type: ImageMeasurementP

  • Y-pos. image meas. 1 (px)
    • Type: ImageMeasurementP

  • Z-pos. image meas. 1 (slice)
    • Type: ImageMeasurementP

  • X-pos. object meas. 1 (px)
    • Type: ObjectMeasurementP

  • Y-pos. object meas. 1 (px)
    • Type: ObjectMeasurementP

  • Z-pos. object meas. 1 (slice)
    • Type: ObjectMeasurementP

  • Reference mode 2
    • Type: ChoiceP
    • Default value: Object centroid
    • Choices:
      • Object centroid
      • Image measurement
      • Object measurement

  • Reference image 2
    • Type: InputImageP

  • X-pos. image meas. 2 (px)
    • Type: ImageMeasurementP

  • Y-pos. image meas. 2 (px)
    • Type: ImageMeasurementP

  • Z-pos. image meas. 2 (slice)
    • Type: ImageMeasurementP

  • X-pos. object meas. 2 (px)
    • Type: ObjectMeasurementP

  • Y-pos. object meas. 2 (px)
    • Type: ObjectMeasurementP

  • Z-pos. object meas. 2 (slice)
    • Type: ObjectMeasurementP

  • Measurement prefix
    • Type: StringP
