Duplicate objects across time

Description

Creates a copy of objects across all frames in the specified image stack.

Parameters

  • Input objects (default = "")

  • Output objects (default = "")

  • Coordinates storage mode (default = "Common")

  • Start frame mode (default = "Fixed value")

  • Start frame fixed value (default = "0")

  • Start frame image (default = "")

  • Start frame image measurement (default = "")

  • Start frame offset (default = "0")

  • End frame mode (default = "Fixed value")

  • End frame fixed value (default = "0")

  • End frame image (default = "")

  • End frame image measurement (default = "")

  • End frame offset (default = "0")
