Measure radial intensity profile

Description

Parameters

  • Input objects
    • Type: InputObjectsP

  • Input image
    • Type: InputImageP

  • Reference mode
    • Type: ChoiceP
    • Default value: Distance from object centroid
    • Choices:
      • Distance from object centroid
      • Distance from object edge
      • Custom distance map

  • Distance map image
    • Type: InputImageP

  • Masking mode
    • Type: ChoiceP
    • Default value: Inside only
    • Choices:
      • Inside and outside
      • Inside only
      • Outside only

  • Number of radial samples
    • Type: IntegerP
    • Default value: 10

  • Range mode
    • Type: ChoiceP
    • Default value: Automatic range
    • Choices:
      • Automatic range
      • Manual range

  • Minimum distance
    • Type: DoubleP
    • Default value: 0.0

  • Maximum distance
    • Type: DoubleP
    • Default value: 1.0
