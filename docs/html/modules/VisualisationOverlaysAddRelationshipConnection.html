Description

Parameters

  • Input image
    • Type: InputImageP

  • Parent objects
    • Type: InputObjectsP

  • Child objects
    • Type: ChildObjectsP

  • Apply to input image
    • Type: BooleanP
    • Default value: false

  • Add output image to workspace
    • Type: BooleanP
    • Default value: false

  • Output image
    • Type: OutputImageP

  • Line width
    • Type: DoubleP
    • Default value: 0.2

  • Render in all frames
    • Type: BooleanP
    • Default value: false

  • Enable multithreading
    • Type: BooleanP
    • Default value: true

  • Colour mode
    • Type: ChoiceP
    • Default value: Single colour
    • Choices:
      • Child count
      • ID
      • Measurement value
      • Parent ID
      • Parent measurement value
      • Random colour
      • Single colour

  • Single colour
    • Type: ChoiceP
    • Default value: White
    • Choices:
      • White
      • Black
      • Red
      • Orange
      • Yellow
      • Green
      • Cyan
      • Blue
      • Violet
      • Magenta

  • Child objects for colour
    • Type: ChildObjectsP

  • Measurement for colour
    • Type: ObjectMeasurementP

  • Parent object for colour
    • Type: ParentObjectsP
