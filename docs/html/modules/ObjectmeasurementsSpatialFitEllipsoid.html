Fit ellipsoid

Description

Parameters

  • Input objects
    • Type: InputObjectsP
  • Template image
    • Type: InputImageP
  • Fitting mode
    • Type: ChoiceP
    • Default value: Fit to surface
    • Choices:
      • Fit to surface
      • Fit to whole
  • Use intensity weighting
    • Type: BooleanP
    • Default value: false
  • Object output mode
    • Type: ChoiceP
    • Default value: Do not store
    • Choices:
      • Do not store
      • Create new objects
      • Update input objects
  • Output objects
    • Type: OutputObjectsP
  • Limit axis length
    • Type: BooleanP
    • Default value: false
  • Maximum axis length
    • Type: DoubleP
    • Default value: 1000.0
