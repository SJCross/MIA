Object measurement calculator

Description

Parameters

  • Input objects
    • Type: InputObjectsP

  • Value mode 1
    • Type: ChoiceP
    • Default value: Measurement
    • Choices:
      • Fixed
      • Image measurement
      • Measurement
      • Object collection statistic

  • Fixed value 1
    • Type: DoubleP
    • Default value: 0.0

  • Image 1
    • Type: InputImageP

  • Image measurement 1
    • Type: ImageMeasurementP

  • Measurement 1
    • Type: ObjectMeasurementP

  • Reference objects 1
    • Type: InputObjectsP

  • Reference measurement 1
    • Type: ObjectMeasurementP

  • Statistic mode 1
    • Type: ChoiceP
    • Default value: Minimum
    • Choices:
      • First value
      • Last value
      • Minimum
      • Mean
      • Maximum
      • Range
      • Standard deviation
      • Sum

  • Value mode 2
    • Type: ChoiceP
    • Default value: Measurement
    • Choices:
      • Fixed
      • Image measurement
      • Measurement
      • Object collection statistic

  • Fixed value 2
    • Type: DoubleP
    • Default value: 0.0

  • Image 2
    • Type: InputImageP

  • Image measurement 2
    • Type: ImageMeasurementP

  • Measurement 2
    • Type: ObjectMeasurementP

  • Reference objects 2
    • Type: InputObjectsP

  • Reference measurement 2
    • Type: ObjectMeasurementP

  • Statistic mode 2
    • Type: ChoiceP
    • Default value: Minimum
    • Choices:
      • First value
      • Last value
      • Minimum
      • Mean
      • Maximum
      • Range
      • Standard deviation
      • Sum

  • Output measurement
    • Type: StringP

  • Calculation mode
    • Type: ChoiceP
    • Default value: Add measurement 1 and measurement 2
    • Choices:
      • Add measurement 1 and measurement 2
      • Divide measurement 1 by measurement 2
      • Multiply measurement 1 and measurement 2
      • Subtract measurement 2 from measurement 1
