Measure intensity distribution

Description

CURRENTLY ONLY WORKS IN 3D

Parameters

  • Input image
    • Type: InputImageP
  • Measurement type
    • Type: ChoiceP
    • Default value: Distance profile
    • Choices:
      • Distance profile
      • Fraction proximal to objects
      • Intensity-weighted proximity
  • Distance map image
    • Type: InputImageP
  • Number of radial samples
    • Type: IntegerP
    • Default value: 10
  • Range mode
    • Type: ChoiceP
    • Default value: Automatic range
    • Choices:
      • Automatic range
      • Manual range
  • Minimum distance
    • Type: DoubleP
    • Default value: 0.0
  • Maximum distance
    • Type: DoubleP
    • Default value: 1.0
  • Save profile mode
    • Type: ChoiceP
    • Default value: Don't save
    • Choices:
      • Don't save
      • Individual files
  • Profile file suffix
    • Type: StringP
  • Input objects
    • Type: InputObjectsP
  • Proximal distance
    • Type: DoubleP
    • Default value: 2.0
  • Spatial units
    • Type: ChoiceP
    • Default value: Pixel
    • Choices:
      • Calibrated
      • Pixel
  • Ignore values on objects
    • Type: BooleanP
    • Default value: false
  • Edge distance mode
    • Type: ChoiceP
    • Default value: Inside and outside
    • Choices:
      • Inside and outside
      • Inside only
      • Outside only
