Weka probability maps

Description

Loads a saved WEKA classifier model and applies it to the input image. Returns the multi-channel probability map

Parameters

  • Input image
    • Type: InputImageP

  • Convert to RGB
    • Type: BooleanP
    • Default value: false

  • Output image
    • Type: OutputImageP

  • Output bit depth
    • Type: ChoiceP
    • Default value: 32
    • Choices:
      • 8
      • 16
      • 32

  • Output single class
    • Type: BooleanP
    • Default value: false

  • Output class
    • Type: IntegerP
    • Default value: 1

  • Classifier file path
    • Type: FilePathP

  • Block size (simultaneous slices)
    • Type: IntegerP
    • Default value: 1
