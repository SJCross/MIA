Conditional analysis termination

Description

Parameters

  • Test mode
    • Type: ChoiceP
    • Default value: Image measurement
    • Choices:
      • Image measurement
      • Object count

  • Input image
    • Type: InputImageP

  • Input objects
    • Type: InputObjectsP

  • Reference image measurement mode
    • Type: ChoiceP
    • Default value: Terminate if measurement less than
    • Choices:
      • Terminate if measurement less than
      • Terminate if measurement greater than

  • Reference image measurement
    • Type: ImageMeasurementP

  • Reference object count mode
    • Type: ChoiceP
    • Default value: Fewer objects than
    • Choices:
      • Fewer objects than
      • More objects than
      • No objects

  • Reference value
    • Type: DoubleP
    • Default value: 0.0

  • Remove images from workspace
    • Type: BooleanP
    • Default value: false

    • Type: BooleanP
