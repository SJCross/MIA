Normalise intensity

Description

Sets the intensity to maximise the dynamic range of the image. "Clipping fraction" is the fraction of pixels at either end of the range that gets clipped.The "Per object" region mode will normalise all pixels within each object.

Parameters

  • Input image (default = "")

  • Apply to input image (default = "true")

  • Output image (default = "")

  • Region mode (default = "Entire image")

  • Input objects (default = "")

  • Calculation mode (default = "Fast")

  • Clipping fraction (min) (default = "0.0")

  • Clipping fraction (max) (default = "0.0")

  • Minimum range value (default = "0.0")

  • Maximum range value (default = "255.0")
