Unwarp images (manual)

Description

Uses the bUnwarpJ_ plugin by Ignacio Arganda-Carreras. Arganda-Carreras, I., et al., CVAMIA: Computer Vision Approaches to Medical Image Analysis (2006), 85-95

Parameters

  • Input image
    • Type: InputImageP

  • Output image
    • Type: OutputImageP

  • Reference image
    • Type: InputImageP

  • Point selection mode
    • Type: ChoiceP
    • Default value: Select at runtime
    • Choices:
      • Pre-selected points
      • Select at runtime

  • Registration mode
    • Type: ChoiceP
    • Default value: Fast
    • Choices:
      • Fast
      • Accurate
      • Mono

  • Subsample factor
    • Type: IntegerP
    • Default value: 0

  • Initial deformation mode
    • Type: ChoiceP
    • Default value: Very Coarse
    • Choices:
      • Very Coarse
      • Coarse
      • Fine
      • Very Fine

  • Final deformation mode
    • Type: ChoiceP
    • Default value: Fine
    • Choices:
      • Very Coarse
      • Coarse
      • Fine
      • Very Fine
      • Super Fine

  • Divergence weight
    • Type: DoubleP
    • Default value: 0.0

  • Curl weight
    • Type: DoubleP
    • Default value: 0.0

  • Landmark weight
    • Type: DoubleP
    • Default value: 0.0

  • Image weight
    • Type: DoubleP
    • Default value: 1.0

  • Consistency weight
    • Type: DoubleP
    • Default value: 10.0

  • Stop threshold
    • Type: DoubleP
    • Default value: 0.01

  • Fill mode
    • Type: ChoiceP
    • Default value: Black
    • Choices:
      • Black
      • White

  • Enable multithreading
    • Type: BooleanP
    • Default value: true
