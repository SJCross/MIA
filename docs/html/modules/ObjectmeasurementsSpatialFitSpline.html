Fit spline

Description

Parameters

  • Input objects
    • Type: InputObjectsP

  • Object output mode
    • Type: ChoiceP
    • Default value: Do not store
    • Choices:
      • Do not store
      • Individual control points
      • Full contour

  • Output objects
    • Type: OutputObjectsP

  • Export every N points
    • Type: IntegerP
    • Default value: 1

  • Extend to max
    • Type: BooleanP
    • Default value: false

  • Relate to reference point
    • Type: BooleanP
    • Default value: false

  • X-axis reference measurement
    • Type: ObjectMeasurementP

  • Y-axis reference measurement
    • Type: ObjectMeasurementP

  • Spline fitting method
    • Type: ChoiceP
    • Default value: LOESS (smooth fitting)
    • Choices:
      • LOESS (smooth fitting)
      • Standard (fits all points)

  • Number of neighbours (smoothing)
    • Type: IntegerP
    • Default value: 20

  • Iterations
    • Type: IntegerP
    • Default value: 10

  • Accuracy
    • Type: DoubleP
    • Default value: 1.0

  • Measure absolute curvature
    • Type: BooleanP
    • Default value: true

  • Measure signed curvature
    • Type: BooleanP
    • Default value: true

  • Draw spline
    • Type: BooleanP
    • Default value: false

  • Input image
    • Type: InputImageP

  • Apply to image
    • Type: BooleanP
    • Default value: false

  • Output image
    • Type: OutputImageP

  • Line width
    • Type: DoubleP
    • Default value: 1.0

  • Maximum curvature (for colour)
    • Type: DoubleP
    • Default value: 1.0

  • Calculate angle between ends
    • Type: BooleanP
    • Default value: true

  • Fitting range (px)
    • Type: IntegerP
    • Default value: 5
