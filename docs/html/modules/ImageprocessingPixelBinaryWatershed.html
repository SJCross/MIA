Watershed transform

Description

Uses MorphoLibJ implementation of watershed transform.

Parameters

  • Input image (default = "")

  • Apply to input image (default = "true")

  • Output image (default = "")

  • Use markers (default = "false")

  • Input marker image (default = "")

  • Intensity mode (default = "Distance")

  • Intensity image (default = "")

  • Dynamic (default = "1")

  • Connectivity (default = "26")

  • Match Z to XY (default = "true")

  • Enable multithreading (default = "true")
