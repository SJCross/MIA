Fit Gaussian 2D

Description

Gaussian spot fitting. Can take objects as estimated locations. ***Only works in 2D*** ***Only works for refinement of existing spots***

Parameters

  • Input image
    • Type: InputImageP
  • Input objects
    • Type: InputObjectsP
  • Method to estimate spot radius
    • Type: ChoiceP
    • Default value: Fixed value
    • Choices:
      • Fixed value
      • Measurement
  • Radius
    • Type: DoubleP
    • Default value: 1.0
  • Radius measurement
    • Type: ObjectMeasurementP
  • Measurement multiplier
    • Type: DoubleP
    • Default value: 1.0
  • Limit sigma range
    • Type: BooleanP
    • Default value: true
  • Minimum sigma (x Radius)
    • Type: DoubleP
    • Default value: 0.25
  • Maximum sigma (x Radius)
    • Type: DoubleP
    • Default value: 4.0
  • Fixed fitting window
    • Type: BooleanP
    • Default value: false
  • Window size
    • Type: IntegerP
    • Default value: 15
  • Maximum number of evaluations
    • Type: IntegerP
    • Default value: 1000
  • Remove objects with failed fitting
    • Type: BooleanP
    • Default value: false
  • Apply volume
    • Type: BooleanP
    • Default value: true
