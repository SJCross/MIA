Binary operations 2D

Description

Expects black objects on a white background. Performs 2D fill holes, dilate and erode using ImageJ functions.

Parameters

  • Input image (default = "")

  • Apply to input image (default = "true")

  • Output image (default = "")

  • Filter mode (default = "Dilate")

  • Number of iterations (default = "1")

  • Count (default = "1")
