Measure object centroid

Description

Measure mean XYZ centroid for all objects in an object colleciton.
br>Note: Z-coordinates are specified in terms of slices (not pixels).

Parameters

  • Input objects
    • Description: Objects to measure mean XYZ centroid for. Measurements will be associated with each object.
    • Type: InputObjectsP
