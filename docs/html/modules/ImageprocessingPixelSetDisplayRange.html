Set intensity display range

Description

Parameters

  • Image input/output
    • Type: ParamSeparatorP
  • Input image
    • Type: InputImageP
  • Apply to input image
    • Type: BooleanP
    • Default value: true
  • Output image
    • Type: OutputImageP
  • Intensity range
    • Type: ParamSeparatorP
  • Calculation mode
    • Type: ChoiceP
    • Default value: Fast
    • Choices:
      • Fast
      • Manual
      • Precise
  • Clipping fraction
    • Type: DoubleP
    • Default value: 0.0
  • Minimum range value
    • Type: DoubleP
    • Default value: 0.0
  • Maximum range value
    • Type: DoubleP
    • Default value: 255.0
