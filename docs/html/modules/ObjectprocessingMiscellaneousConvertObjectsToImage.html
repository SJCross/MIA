Convert objects to image

Description

Parameters

  • Conversion mode
    • Type: ChoiceP
    • Default value: Objects to image
    • Choices:
      • Image to objects
      • Objects to image

  • Input image
    • Type: InputImageP

  • Output objects
    • Type: OutputObjectsP

  • Volume type
    • Type: ChoiceP
    • Default value: Pointlist
    • Choices:
      • Octree
      • Pointlist
      • Quadtree

  • Template image
    • Type: InputImageP

  • Input objects
    • Type: InputObjectsP

  • Output image
    • Type: OutputImageP

  • Output mode
    • Type: ChoiceP
    • Default value: Whole object
    • Choices:
      • Object centroid
      • Whole object

  • Colour mode
    • Type: ChoiceP
    • Default value: Single colour
    • Choices:
      • Child count
      • ID
      • Measurement value
      • Parent ID
      • Parent measurement value
      • Random colour
      • Single colour

  • Child objects for colour
    • Type: ChildObjectsP

  • Measurement
    • Type: ObjectMeasurementP
