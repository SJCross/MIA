Flip stack

Description

Flips the order of slices in stack. This operation can be performed on the channel, time or Z axis.

Parameters

  • Input image
    • Type: InputImageP
  • Apply to input image
    • Type: BooleanP
    • Default value: true
  • Output image
    • Type: OutputImageP
  • Axis mode
    • Type: ChoiceP
    • Default value: Z
    • Choices:
      • Channel
      • Time
      • Z
