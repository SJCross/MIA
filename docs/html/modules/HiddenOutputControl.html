Description

Parameters

  • Save location
    • Type: ChoiceP
    • Default value: Save with input file
    • Choices:
      • Save with input file
      • Specific location

  • File path
    • Type: FolderPathP

  • Save name mode
    • Type: ChoiceP
    • Default value: Match input file/folder name
    • Choices:
      • Match input file/folder name
      • Specific name

  • File name
    • Type: StringP

  • Export mode
    • Type: ChoiceP
    • Default value: All together
    • Choices:
      • All together
      • Group by metadata
      • Individual files
      • None

  • Metadata item for grouping
    • Type: MetadataItemP

  • Continuous data export
    • Type: BooleanP
    • Default value: false

  • Save every n files
    • Type: IntegerP
    • Default value: 10

  • Append date/time mode
    • Type: ChoiceP
    • Default value: Never
    • Choices:
      • Always
      • If file exists
      • Never

  • Export summary
    • Type: BooleanP
    • Default value: true

  • Summary mode
    • Type: ChoiceP
    • Default value: Per input file
    • Choices:
      • Per input file
      • Per timepoint per input file
      • Group by metadata

  • Metadata item for summary
    • Type: MetadataItemP

  • Show object counts
    • Type: BooleanP
    • Default value: true

  • Export individual objects
    • Type: BooleanP
    • Default value: true
