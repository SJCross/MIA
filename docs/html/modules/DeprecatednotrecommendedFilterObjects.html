Filter objects

Description

Parameters

  • Input objects
    • Type: InputObjectsP

  • Filter mode
    • Type: ChoiceP
    • Default value: Remove filtered objects
    • Choices:
      • Do nothing
      • Move filtered objects to new class
      • Remove filtered objects

  • Output (filtered) objects
    • Type: OutputObjectsP

  • Method for filtering
    • Type: ChoiceP
    • Default value: Exclude objects on image edge (2D)
    • Choices:
      • Exclude objects on image edge (2D)
      • Remove objects with missing measurements
      • Remove objects without parent
      • Remove objects with a parent
      • Remove objects with fewer children than:
      • Remove objects with more children than:
      • Remove objects with measurements < than:
      • Remove objects with measurements > than:
      • Specific object IDs (runtime)

  • Reference image
    • Type: InputImageP

  • Include Z-position
    • Type: BooleanP
    • Default value: false

  • Measurement to filter on
    • Type: ObjectMeasurementP

  • Parent object
    • Type: ParentObjectsP

  • Child objects
    • Type: ChildObjectsP

  • Reference mode
    • Type: ChoiceP
    • Default value: Fixed value
    • Choices:
      • Fixed value
      • Image measurement
      • Parent object measurement

  • Reference value
    • Type: DoubleP
    • Default value: 1.0

  • Reference value image
    • Type: InputImageP

  • Reference image measurement
    • Type: ImageMeasurementP

  • Reference value parent object
    • Type: ParentObjectsP

  • Reference object measurement
    • Type: ObjectMeasurementP

  • Reference value multiplier
    • Type: DoubleP
    • Default value: 1.0

  • Show image
    • Type: BooleanP
    • Default value: true

  • Image to display
    • Type: InputImageP
