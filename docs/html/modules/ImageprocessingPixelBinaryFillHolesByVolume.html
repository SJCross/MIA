Fill holes by volume

Description

Parameters

  • Input image
    • Type: InputImageP

  • Apply to input image
    • Type: BooleanP
    • Default value: true

  • Output image
    • Type: OutputImageP

  • Black-white mode
    • Type: ChoiceP
    • Default value: Fill white holes
    • Choices:
      • Fill black holes
      • Fill white holes

  • Use minimum volume
    • Type: BooleanP
    • Default value: true

  • Minimum size
    • Type: DoubleP
    • Default value: 0.0

  • Use maximum volume
    • Type: BooleanP
    • Default value: true

  • Maximum size
    • Type: DoubleP
    • Default value: 1000.0

  • Calibrated units
    • Type: BooleanP
    • Default value: false

  • Enable multithreading
    • Description: Break the image down into strips, each one processed on a separate CPU thread. The overhead required to do this means it's best for large multi-core CPUs, but should be left disabled for small images or on CPUs with few cores.
    • Type: BooleanP
    • Default value: true

  • Minimum strip width
    • Description: Minimum width of each strip to be processed on a separate CPU thread. Measured in pixel units.
    • Type: IntegerP
    • Default value: 60
