Focus stack

Description

Channels will be focused individually. Uses the StackFocuser plugin created by Mikhail Umorin. Source downloaded from https://imagej.nih.gov/ij/plugins/download/Stack_Focuser_.java on 06-June-2018

Parameters

  • Input image
    • Type: InputImageP
  • Output focused image
    • Type: OutputImageP
  • Use existing height image
    • Type: BooleanP
    • Default value: false
  • Input height image
    • Type: InputImageP
  • Range
    • Type: IntegerP
    • Default value: 11
  • Smooth height map
    • Type: BooleanP
    • Default value: true
  • Add height map image to workspace
    • Type: BooleanP
    • Default value: false
  • Output height image
    • Type: OutputImageP
  • Show height image
    • Type: BooleanP
    • Default value: false
