Ridge detection

Description

Uses the RidgeDetection Fiji plugin by Thorsten Wagner, which implements Carsten Steger's paper "An Unbiased Detector of Curvilinear Structures" INCOMPLETE

Parameters

  • Input image (default = "")

  • Output objects (default = "")

  • Contour contrast (default = "Dark line")

  • Lower threshold (default = "0.5")

  • Upper threshold (default = "0.85")

  • Sigma (default = "3.0")

  • Calibrated units (default = "false")

  • Extend line (default = "false")

  • Estimate width (default = "false")

  • Minimum length (default = "0.0")

  • Maximum length (default = "0.0")

  • Link contours (default = "false")

  • Limit end misalignment (default = "false")

  • Alignment range (px) (default = "3")

  • Maximum end misalignment (degs) (default = "10.0")
