Merge channels

Description

NOTE: This Module has been superseeded by the more generalised "Concatenate stacks" Module. It will be removed in a future release. Combines image stacks as different channels. Output is automatically converted to a composite image.

Parameters

  • Add image
    • Type: ParameterGroup

  • Overwrite mode
    • Type: ChoiceP
    • Default value: Create new image
    • Choices:
      • Create new image
      • Overwrite image

  • Output image
    • Type: OutputImageP

  • Image index to overwrite (>= 1)
    • Type: IntegerP
    • Default value: 1
