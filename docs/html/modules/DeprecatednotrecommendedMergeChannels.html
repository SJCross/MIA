Merge channels

Description

NOTE: This Module has been superseeded by the more generalised "Concatenate stacks" Module. It will be removed in a future release. Combines image stacks as different channels. Output is automatically converted to a composite image.

Parameters

  • Add image (default = "")

  • Overwrite mode (default = "Create new image")

  • Output image (default = "")

  • Image index to overwrite (>= 1) (default = "1")
